# geo-nyc

The Next.js app lives in **`apps/web`**. From the repo root:

```bash
cd apps/web
npm install
npm run dev
```

See **`apps/web/README.md`** for map data, env vars, and the mask script.

**New repo on GitHub:** create an empty repo, then from `geo-nyc/`:

```bash
git init
git add .
git commit -m "geoNYC web app"
git branch -M main
git remote add origin https://github.com/<you>/<repo>.git
git push -u origin main
```

Replace the remote URL with yours.
